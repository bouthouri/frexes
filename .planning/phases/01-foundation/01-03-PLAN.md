    1. Create `public/favicon.svg` — a bold geometric "F" lettermark in a sharp square container:
       ```svg

         <rect width="32" height="32" fill="#000000" rx="4"/>
         <text x="8" y="24" font-family="Inter, system-ui, sans-serif" font-weight="800" font-size="22" fill="#FFFFFF">F</text>
       </svg>
       ```
       Per locked decision: Pure black (#000000) background, white (#FFFFFF) letter, sharp square container.
       The font-family matches the heading font for cohesive brand identity (locked decision).

       NOTE: If the SVG text element doesn't render cleanly at 32x32, use a path-based F instead of text. The key constraints are: geometric, bold, black square with white F, sharp corners (rx=4 max).

    2. Create `public/favicon.png` — 32x32 PNG version. Since we can't generate PNG directly, create a simple script or use the SVG as the primary favicon. For the PNG, create a minimal placeholder and note it should be exported from the SVG for pixel-perfect rendering.

       Approach: Create a small Bun script to convert SVG to PNG if sharp/canvas is available, otherwise create a base64-encoded 32x32 PNG with the same design. The simplest path:
       ```bash
       bun add -d sharp
       ```
       Then create a one-off script `scripts/generate-favicon.ts`:
       ```ts
       import sharp from "sharp";
       import { readFile } from "fs/promises";

       const svg = await readFile("public/favicon.svg");
       await sharp(svg).resize(32, 32).png().toFile("public/favicon.png");
       console.log("favicon.png generated");
       ```
       Run it: `bun scripts/generate-favicon.ts`
       Then remove sharp from devDeps if not needed: `bun remove sharp`

       If sharp doesn't work on this system, embed a hand-crafted 32x32 PNG (base64) or skip the PNG and rely on SVG favicon (all modern browsers support it). Note the gap for manual export.

    3. Create `src/layouts/BaseLayout.astro`:
       ```astro

         title = "Frexes",
         description = "Building apps that actually help you get things done",
       } = Astro.props;
       ---
       <!doctype html>
       <html lang="en">
         <head>
           <meta charset="utf-8" />
           <meta name="viewport" content="width=device-width, initial-scale=1" />
           <meta name="description" content={description} />
           <meta name="generator" content={Astro.generator} />
           <link rel="icon" href="/favicon.svg" type="image/svg+xml" />
           <link rel="icon" href="/favicon.png" type="image/png" sizes="32x32" />
           <title>{title}</title>
         </head>
         <body class="bg-background text-foreground font-sans antialiased">
           <slot />
         </body>
       </html>
       ```
